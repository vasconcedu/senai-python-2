# Setup 

## Como configurar o programa `gastos.py`?

1. Baixe o arquivo `categorias_gastos.txt`;
2. Baixe o arquivo `gastos.json`;
3. Baixe o arquivo `gastos.py`;
4. Salve os 3 arquivos acima no mesmo diretório;
5. Instale o pacote `streamlit`, executando o comando abaixo no Prompt de Comando do Windows (Menu Iniciar > CMD):

    ```
    pip install streamlit
    ```

6. Abra os arquivos no VS Code;
7. Abra um terminal no VS Code (Terminal > New Terminal);
8. Se necessário, navegue até a pasta onde você salvou os arquivos, usando o comando `cd`;
9. Execute o comando abaixo no terminal para iniciar a aplicação:

    ```
    python -m streamlit run gastos.py
    ```
